## 2. Main Screen UI

### Layout Structure

* A top AppBar (Toolbar)
* Below it, a horizontal section with **three selectable bars**:

  * "Folder"
  * "Files"
  * "Playlist"

### Behavior of Bars

* Only one bar is **active/selected** at a time
* Bars support:

  * **Swipe left/right gestures**
  * **Left arrow button (←)**
  * **Right arrow button (→)**

### Interaction Logic

#### Folder Bar

* Displays current folder
* On selection: plays all MP3 files from current folder
* On right swipe / right arrow:

  * Move to next sibling folder
  * Load and play MP3s from that folder
* On left swipe / left arrow:

  * Move to previous sibling folder

#### Files Bar

* Displays current MP3 file
* On selection: plays that file
* On right swipe / right arrow:

  * Move to next MP3 file in folder
  * If last file reached:

    * Move to next sibling folder
    * Play first MP3 in that folder
* On left swipe / left arrow:

  * Move to previous MP3 file

#### Playlist Bar

* Displays current playlist
* On selection: plays playlist
* On right swipe / right arrow:

  * Move to next playlist
* On left swipe / left arrow:

  * Move to previous playlist
